Must / Mustn’t — обов’язок та заборона

Коли потрібно щось зробити і коли це заборонено.

Positive You must wear a seat belt.
Negative You must not (mustn’t) smoke here.
Questions Must we finish today? — Yes, we must. / No, we don’t have to. (розмовне)
Use it right!
Must = strong obligation from the speaker.
Mustn’t = prohibition.
Для відсутності необхідності частіше використовуємо don’t have to.
You mustn’t go if you don’t want. ✗ → You don’t have to go if you don’t want. ✓
