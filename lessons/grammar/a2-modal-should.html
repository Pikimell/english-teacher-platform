Should / Shouldn’t — Advice

Використовуємо should для порад і рекомендацій.

Форма та структура
Should + base verb (advice). Shouldn’t + base verb.
Коли вживати
Поради: You should see a doctor. Критика або попередження: You shouldn’t stay up so late.
Приклади Заперечення / Варіації
  • You should try the local food.
  • He should apologise to her.
  • You shouldn't drink so much coffee.
  • They shouldn't ignore that email.
Практика
Гра “Problem & Advice”: один студент описує проблему, інший дає дві поради зі should/shouldn’t.
