Кольори (Colours / Colors)

Лексика про кольори потрібна у щоденному житті та на роботі: опис дизайну, інтерфейсів, документів, одягу, інтер’єру. Нижче — базова палітра, часті відтінки та фрази, які допоможуть чітко формулювати запити й коментарі в дорослому контексті.

Базові кольори

Мінімальний набір для комфортного спілкування

Англійською Переклад Приклад у реченні

red червоний Red buttons usually signal an alert. — Червоні кнопки зазвичай позначають тривогу.
blue синій Our brand uses a dark blue palette. — Наш бренд використовує темно-синю палітру.
green зелений Green indicates success or confirmation. — Зелений означає успіх або підтвердження.

Як коректно просити про зміни кольору

Короткі, зрозумілі формулювання для задач у роботі

Ситуація/намір Приклад фрази (EN) Переклад
Попросити змінити відтінок Could we make the buttons a bit darker? Можемо зробити кнопки трохи темнішими?
Зменшити яскравість Please use a more muted yellow. Будь ласка, використай більш приглушений жовтий.
Уніфікація з бренд-гайдом Let’s align the icons with the brand blue. Давай вирівняємо іконки під бренд-синій.
Контраст і читабельність This text needs higher contrast against the background. Цьому тексту потрібен вищий контраст із фоном.
Альтернатива Alternatively, try a light grey background. Альтернативно, спробуй світло-сірий фон.

Практичні нотатки

Для листування, тасків і стендапів

Теза Коментар Приклад (EN)
Уточнюй відтінок “Синій” занадто загально; додай опис: dark, light, muted тощо. Use a dark, muted blue for the header.
Контекст використання Кнопка, фон, текст — вимоги різні. Keep links blue, but make the body text black.
Послідовність Один і той самий відтінок у схожих елементах. Apply the same grey to all captions.

Короткі моделі фраз

Щоб говорити чітко й доросло

Шаблон Приклад (EN) Коли доречно
[Make/Keep] it + light/dark/muted/bright + [colour] Keep it light grey. Налаштування відтінку елемента
[Switch/Change] to + colour Switch to dark blue for the header. Запит на заміну кольору
[Increase/Reduce] the contrast Reduce the contrast on hover. Доступність, наведення курсора
Use a more/less + adjective + colour Use a more muted orange. Точне коригування тону
