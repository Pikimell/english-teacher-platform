First Conditional — if + Present, will + V

Реальні умови у майбутньому, варіанти з модальними

IF – CLAUSE
(hypothesis) 		MAIN CLAUSE
(result) 		IF – CLAUSE
(hypothesis) 		MAIN CLAUSE
(result)
If + present simple, will + V If + present simple, modal verb + V
- If Jane gets a salary, she will buy a ticket to the concert of her favourite band.
- If she learns the lyrics of their songs, she will sing along with them at the concert.
- If she has a chance, she will ask them for autographs. 		- If you start jogging, you can lose your weight.
- If you train hard, you can take part in a running competition.
- If you take part in a competition, you may win.
Use it right!
Don’t forget about a comma.
We use it before the main clause.
e.g.: If I have enough time, I’ll help you tomorrow.

BUT there is no comma before the if-clause
e.g.: I’ll help you tomorrow if I have enough time.
