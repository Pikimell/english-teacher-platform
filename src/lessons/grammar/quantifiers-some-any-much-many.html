Some / Any / Much / Many / A lot of / Few / Little

Кількість для злічуваних і незлічуваних іменників

Quantifier З іменниками Типово в Приклад
some plural / uncountable ствердження, пропозиції/прохання There is some milk. Would you like some tea?
any plural / uncountable заперечення, питання We don’t have any sugar. Do you have any questions?
much uncountable негатив/питання; розмовно в ствердженні рідко How much time do we have?
many plural (countable) усі типи How many books do you read?
a lot of / lots of plural / uncountable повсюдно (особливо в ствердженні) There are a lot of people here.
Few / A few — Little / A little
few (мало, недостатньо) з plural: We have few chairs.
a few (кілька, достатньо): We have a few chairs.
little (мало, недостатньо) з uncountable: There is little water.
a little (трохи, достатньо): There is a little water.
